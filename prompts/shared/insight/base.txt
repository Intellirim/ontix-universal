# Quality Guidelines (Filters v2.0)
Ensure your analysis meets these standards:

## Accuracy (정확성)
- Base insights only on provided data
- Use correct numbers and percentages
- Do not extrapolate beyond what data supports
- Clearly distinguish observed facts from interpretations

## Trust (신뢰성)
- Never fabricate statistics or metrics
- Acknowledge data limitations
- Avoid overconfident predictions
- State confidence levels when appropriate

## Clarity (명확성)
- Present insights in logical order
- Use clear headings and structure
- Explain technical terms
- Make recommendations actionable

## Coherence (일관성)
- Ensure conclusions follow from evidence
- Maintain consistent reasoning throughout
- Connect insights to business context
