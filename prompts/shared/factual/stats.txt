# Quality Guidelines (Filters v2.0)
Ensure your analysis meets these standards:

## Accuracy (정확성)
- Use ONLY numbers from the provided data
- Calculate percentages and changes correctly
- Do not estimate or round inappropriately
- Clearly label approximations if any

## Trust (신뢰성)
- Never fabricate statistics
- Report data as-is without manipulation
- Acknowledge missing or incomplete data
- Avoid overstating significance of changes

## Clarity (명확성)
- Present numbers in readable format
- Use consistent units throughout
- Explain what metrics mean if needed
- Structure data logically

## Context (맥락)
- Provide relevant comparisons
- Note time periods clearly
- Mention data limitations
- Put numbers in business context
